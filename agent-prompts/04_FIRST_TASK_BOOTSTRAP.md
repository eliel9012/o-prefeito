# Primeira tarefa recomendada ao agente

Use com Kimi K2.5 ou Kimi K2.7 Code:

```text
Leia AGENTS.md, PROJECT_BRIEF.md, CURRENT_MILESTONE.md, TASKS.md e CHANGELOG_AGENT.md.

Tarefa: faça somente o rebranding mínimo inicial do projeto para "O Prefeito: Simulador de Cidade Brasileira".

Escopo permitido:
- atualizar título principal, metadata e textos visíveis principais;
- adicionar um subtítulo em português;
- não alterar simulação;
- não alterar renderização de cidade;
- não adicionar Tauri ainda;
- não adicionar assets gerados por IA ainda.

Ao final:
- rode npm run build;
- atualize TASKS.md;
- atualize CHANGELOG_AGENT.md;
- explique como testar manualmente.
```
