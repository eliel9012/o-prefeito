# Current Milestone

## Milestone atual: MVP brasileiro funcional

Objetivo: transformar o IsoCity em uma versão funcional chamada **O Prefeito: Simulador de Cidade Brasileira**, mantendo o jogo abrindo e compilando.

## Entregáveis deste milestone

1. Rebranding mínimo:
   - nome do app;
   - título/tela inicial;
   - textos principais em PT-BR.

2. Sistema municipal simples:
   - dinheiro municipal;
   - receita mensal;
   - despesas mensais;
   - popularidade;
   - alertas de eventos.

3. Eventos brasileiros iniciais:
   - buraco na via;
   - enchente;
   - licitação atrasada;
   - pressão da Câmara;
   - recomendação do Ministério Público.

4. Build:
   - `npm run build` funcionando;
   - plano Tauri preparado;
   - build macOS testável no Mac.

## Fora do escopo deste milestone

- novos sprites definitivos;
- simulação avançada de trânsito;
- sistema eleitoral;
- campanhas partidárias;
- DLCs;
- multiplayer;
- IA generativa dentro do jogo.
