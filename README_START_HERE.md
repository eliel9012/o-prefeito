# O Prefeito: Simulador de Cidade Brasileira — Agent Docs

Este pacote foi criado para guiar agentes de IA na transformação do fork do projeto `amilich/isometric-city` em um jogo brasileiro, funcional, empacotável para macOS e futuramente comercializável no Brasil.

## Nome de trabalho recomendado

**O Prefeito: Simulador de Cidade Brasileira**

Motivos:
- `O Prefeito` é curto, memorável e forte para marca.
- O subtítulo melhora SEO: `simulador`, `cidade`, `brasileira`.
- Evita depender da marca `SimCity`, que deve ser usada apenas como referência interna de gênero.

## Objetivo do MVP

Criar uma versão jogável baseada no IsoCity com:
- interface em português brasileiro;
- identidade visual brasileira;
- orçamento municipal simples;
- impostos brasileiros simplificados, como IPTU e ISS;
- eventos brasileiros, como enchente, buraco, obra parada, licitação, pressão popular e Ministério Público;
- build web local;
- plano de empacotamento para macOS via Tauri.
